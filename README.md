# FB Pro Blaster Build Repository

Production build repository for **FB Pro Blaster**.

This repository is used to publish release artifacts and downloadable production packages for operators and customers. Source code development happens in the main project repository, while this repository is focused on versioned build files and release delivery.

## What you can find here

- Production ZIP builds
- Versioned release artifacts
- Downloadable packages for deployment

## Notes

- This repository is for build distribution, not active source development.
- Runtime secrets and operator-specific config should not be committed to production artifacts.
- If you need setup guidance after download, use the included documentation inside the build package or the official landing page.
